# ER Mock App (WIP)

## Setup Instructions

1. Create a `.env.local` file in the base directory and add:

   ```
   REACT_APP_OPENAI_API_KEY=your-api-key
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the development server:

   ```bash
   npm start
   ```

## Requirements

- Node.js
- OpenAI API key
